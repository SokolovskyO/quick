# Quick — ИИ-помощник для текста на Windows

**Выдели текст в любом приложении → нажми Ctrl+Shift+A → получи результат за секунды.**

👉 [quick-app.tech](https://quick-app.tech)

---

## Что это

Quick — утилита для Windows, которая живёт в системном трее и помогает работать с текстом в любом приложении. Браузер, Word, Telegram, Notion, почта — везде одинаково.

Работает на базе GLM-5 от Zhipu AI.

---

## Возможности

| Действие | Что делает |
|---|---|
| Орфография | Исправляет ошибки, не меняя стиль |
| Запятые | Расставляет пунктуацию без изменения слов |
| Перевод | На русский или английский |
| Перефраз | Переформулирует, сохраняя смысл |
| Объяснить | Объясняет слово или фразу простым языком |

## Установка

### 1. Установи Python
Скачай с [python.org](https://python.org/downloads/) — при установке поставь галку **"Add Python to PATH"**

### 2. Скачай Quick
```
git clone https://github.com/SokolovskyO/quick.git
cd quick
```
Или просто скачай ZIP и распакуй.

### 3. Запусти
Дважды кликни на **`ЗАПУСТИТЬ.bat`** — он сам установит все зависимости.

### 4. Введи API-ключ
Иконка в трее → **Настройки** → вставь ключ от GLM.

Получить ключ: [bigmodel.cn](https://bigmodel.cn)

---

## Как пользоваться

1. Выдели любой текст в любом приложении
2. Нажми **Ctrl + Shift + A**
3. Выбери действие из меню
4. Нажми **Копировать** или вставь результат обратно

---
